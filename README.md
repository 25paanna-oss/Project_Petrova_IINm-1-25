# Project_Petrova_IINm-1-25
Программа перевод из голосового сообщения в аудио, а также озвучивает текст

Проект использует две модели:
- Whisper-small (ASR)
- Silero TTS (озвучка текста)

Этапы:
1. Speech-to-Text на аудиофайле
2. Text-to-Speech из распознанного текста
3. Возможность озвучивать кастомный текст

Файл `Проект_Петрова.ipynb` содержит рабочий код первого этапа проекта.
